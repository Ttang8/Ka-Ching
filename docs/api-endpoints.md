# API Endpoints

### Users

- `POST /api/users`
- `PATCH /api/users`

### Session

- `POST /api/session`
- `DELETE /api/session`

### Items

- `GET /api/items`
- `POST /api/items`
- `DELETE /api/items/:id`

### Interests

- `POST /api/interests`
- `DELETE /api/interests/:id`

### Categories

- `GET /api/categories`
- `GET /api/categories/:id`

### Feedback

- `GET /api/feedback`
- `POST /api/feedback`
- `GET /api/feedback/:id`
- `DELETE /api/feedback/:id`
